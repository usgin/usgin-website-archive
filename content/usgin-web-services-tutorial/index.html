USGIN Web Services Tutorial

The State Geothermal Data (SGD) [Dead Link, http://www.stategeothermaldata.org] project is an implementation of the Geoscience Information Network (GIN) model for a distributed, interoperable network of geologic data. SGD implements the rules, protocols and standards of GIN.

To facilitate the creation of a distributed and interoperable network, GIN and SGD promote the use of web services to host geologic data.

If you are a subrecipient under the Arizona Geological Survey for the Department of Energy (DOE) National Geothermal Data System (NGDS), and if you have data to submit, then it is possible for you to publish your data as a web service.
