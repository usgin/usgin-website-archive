Welcome to the USGIN Tutorial Page

The United States Geoscience Information Network (USGIN) is a large-scale initiative that mobilizes a number of potentially difficult concepts. In addition, some USGIN procedures can be technical and demanding. This page provides a list of tutorials to describe those concepts and procedures that are most likely to prove challenging.

Though all of the tutorials below reflect USGIN policy, some are more applicable to the State Geothermal Data (SGD) project.

Funding for the USGIN initiative provided by the National Science Foundation http://nsf.gov under NSF award number EAR-0753154, as part of the INTEROP initiative. The USGIN initiative is managed by the Arizona Geological Survey (AZGS) [Dead Link, http://azgs.az.gov] on behalf of the Association of American State Geologists (AASG). USGIN active partners include GEON and EarthChem.
