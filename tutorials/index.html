USGIN Tutorials

This page provides a list of tutorials to describe USGIN concepts and procedures that are most likely to prove challenging.
Tutorial Description Web PDF Video
Data Delivery Cycle Tutorial

This tutorial describes the typical AASG State Geothermal Data data delivery cycle and the manner in which the NGDS Document Repository [Dead Link, http://repository.stategeothermaldata.org/] facilitates that process. The AASG Geothermal Data Project provides data for the National Geothermal Data System (NGDS) [Dead Link, http://geothermaldata.org/]. For more information about the repository, see the NGDS Geothermal Data Repository tutorial.

 N/A PDF N/A
GeoSciML-Portrayal Cookbook

The GeoSciML-Portrayal Cookbook is a tutorial that introduces the concept of the GeoSciML-Portrayal XML schema and provides instructions for web service deployment for data mapped to the GeoSciML-Portrayal schema. This document is primarily useful to data providers who are interested in creating  NGDS web services.

 N/A PDF N/A
NGDS Content Model Tutorial

A tutorial describing National Geothermal Data System (NGDS) [Dead Link, http://geothermaldata.org/] content models and their use. NGDS content models provide users with schemas that are designed to increase the interoperability of any data that has been mapped to them; different schemas exist for different kinds of data.

 Web PDF Video
NGDS Data Explorer Tutorial

A tutorial providing instructions regarding the access to and usage of the NGDS Data Explorer [Dead Link, http://data.geothermaldatasystem.org/] web application. This web-accessible utility is designed to help users browse, discover, and access National Geothermal Data Syetms (NGDS) [Dead Link, http://geothermaldata.org/] web feature services. In addition, this application allows users to easily download the attributes of NGDS web feature services.

 Web PDF Video
NGDS Geothermal Data Repository Tutorial

This tutorial describes the usage of the National Geothermal Data System (NGDS) [Dead Link, http://geothermaldata.org/] Geothermal Data Repository [Dead Link, http://repository.stategeothermaldata.org/repository/]. The NGDS Geothermal Data Repository is a web-accessible utility that allows registered users to upload files and create and manage metadata records; proper metadata records are necessary to locate and access NGDS data.

NGDS Naming Conventions

This document describes the conventions used when naming National Geothermal Data System (NGDS) [Dead Link, http://geothermaldata.org/] web services. This tutorial is primarily intended for NGDS subrecipients who are submitting data to NGDS in the form of web services structured by NGDS schemas (which can be viewed at http://schemas.usgin.org/ [Dead Link, http://schemas.usgin.org/]). NGDS schemas are provided by NGDS content models; for more information about NGDS content models, see the NGDS Content Model Tutorial.

 Web PDF N/A
USGIN CSW Client

A description of the USGIN CSW Client [Dead Link, http://lab.usgin.org/applications/doc/2nd-version-csw-client-application] ArcGIS AddIn. When installed, this product allows users to search specific catalog services (CSW) from ESRI ArcMap and access any web services returned by the query without leaving ArcMap.

 Web PDF Video
USGIN Metadata Tutorial

A conceptual introduction to metadata and the manner in which metadata is used in the context of USGIN and the National Geothermal Data System (NGDS) [Dead Link, http://geothermaldata.org/].

USGIN URI Tutorial

An introduction to URIs as used within the context of USGIN. USGIN URIs are systematic identifiers that are designed to act as interoperable database keys that permit databases to refer to one another across USGIN. In addition, USGIN URIs are designed to dereference to a web-accessible description of the resource they identify.

 Web PDF N/A
XML Tutorial

An introduction to XML and its usage within the context of USGIN. XML is a markup language that acts as a general-purpose interchange format for USGIN data.
